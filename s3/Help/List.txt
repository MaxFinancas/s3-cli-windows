*s3 list [<bucket>[/<keyprefix>]] [/key:<key> /secret:<secret>]

Example:
s3 list mybucket/pic*

    Lists the keys in the bucket beginning with the keyprefix, if supplied.  A
    trailing asterisk on the keyprefix is ignored.  With no parameters, gets 
    the list of buckets.
    
    /key & /secret let you to pass the Amazon S3 access key and secret key
    on the command line (even if you've previously used auth).
